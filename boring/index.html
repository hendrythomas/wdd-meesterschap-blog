Meesterschap

Ik wil leren om data op nieuwe manieren in websites te implementeren.

  • Ik ga dit doen door lessen te volgen en zelfstandig te studeren.
  • Mijn doel is bereikt als ik meerde APIs werkend heb gekregen.

Ik wil meer unieke layouts maken.

  • Ik ga dit doen door meer onderdelen van HTML en CSS te leren.
  • Mijn doel is bereikt als ik meerdere nieuwe tags/properties heb gebruikt in een website.

Ik wil meer websites online zetten, zodat ik beter word in web development en een groter portfolio heb.

  • Ik ga dit doen door opdrachten te maken en wat ik geleerd heb toe te passen in andere projecten.
  • Dit doel is bereikt als ik een eigen project online heb gezet.

Leerdoelen meesterproef

Ik wil leren hoe ik CSS-animaties gepast toepas op content.

  • Ik ga dit doen door meer onderzoek te doen.
  • Dit doel is bereikt als ik 1 animatie heb getest en op de website gezet.

Ik wil een unieke layout kunnen maken.

  • Ik ga dit doen door de identiteit van de organisatie te begrijpen en verwerken in de layout.
  • Dit doel is bereikt als de layout van de website geïmplementeerd is.

Ik wil een website maken die voor mobiel getest is.

  • Ik ga dit doen door een echte user test uit te voeren.
  • Dit doel is bereikt als ik voldoe aan de bijbehorende eisen.
